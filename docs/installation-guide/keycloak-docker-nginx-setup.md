## Keycloak installation

```sh
docker run -p 8080:8080 -e PROXY_ADDRESS_FORWARDING=true -e KEYCLOAK_USER=admin -e KEYCLOAK_PASSWORD=admin quay.io/keycloak/keycloak:12.0.2
```

## Nginx reverse proxy

```
server {
    listen 80;
    listen [::]:80;
    server_name example.com;

    location / {
        rewrite ^ https://example.com$request_uri? permanent;
    }
}
```
